# scala-snippets #

## General Info ##

sbt project created with giter8: https://github.com/n8han/giter8<br />
giter8 template used: https://github.com/ymasory/sbt.g8


## Create Eclipse Project ##

```sh
$ cd scala-snippets
$ chmod u+x sbt
$ ./sbt
> +eclipse with-source=true
```


## Build & run ##

```sh
$ cd scala-snippets
$ chmod u+x sbt
$ ./sbt
> +run
```
